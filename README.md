# Spray

This repo is created for practicing and understanding 
[Spray not-a-framework](http://spray.io/introduction/what-is-spray/).

### Build & Run ###

```sh
$ cd spray
$ sbt ~run
```

Open [http://localhost:9000/](http://localhost:9000/) in your browser.
